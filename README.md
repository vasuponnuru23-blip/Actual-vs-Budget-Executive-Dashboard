# 📊 Actual-vs-Budget-Executive-Dashboard

1.📌 Project Overview
An interactive Power BI dashboard designed to monitor and compare Actual vs Budget financial performance across projects. This dashboard provides executives and business stakeholders with a centralized view of project income, expenses, budget utilization, and financial variances, enabling faster and more informed decision-making.

2.🎯 Project Purpose
The objective of this project is to transform raw financial data into actionable business insights by developing an executive dashboard that:
- Tracks Actual vs Budget performance.
- Monitors project profitability.
- Identifies budget overruns and cost-saving opportunities.
- Analyzes financial trends across multiple years.
- Supports strategic financial planning through interactive reporting.

3.🛠️ Tech Stack  
- Power BI Desktop
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)
- Data Modeling
- Microsoft Excel (Data Source)
- Matrix Visual
- Clustered Column Charts
- KPI Cards
- Slicers
- Conditional Formatting (Data Bars & Font Colors)
- Dynamic Measures
- Interactive Filtering
- Custom Dashboard Design

4.📂 Data Source
The dataset used in this project was obtained from a YouTube tutorial for learning and educational purposes. The data was imported from Microsoft Excel and transformed using Power Query before being modeled and visualized in Power BI.
Data Format: Microsoft Excel (.xlsx)

5.⭐ Featured & Highlights

1️⃣ Business Problem
Managing project finances using spreadsheets or static reports makes it difficult to quickly identify projects that are exceeding budgets or underperforming. Decision-makers require a centralized dashboard that delivers real-time visibility into financial performance while reducing manual reporting efforts.

2️⃣ Goal of the Dashboard
   The dashboard was designed to:
- Compare Actual vs Budgeted Income.
- Compare Actual vs Budgeted Expenses.
- Monitor project financial performance.
- Track budget utilization.
- Identify projects exceeding or falling below budget.
- Enable executive-level decision making.
- Provide interactive filtering by Year, Month and Project ID.

3️⃣ Walkthrough of Key Visuals (Displays)

💰 Income KPI
- Income Actual
- Income Budget
- Income Variance %
- Percentage of Income Budget Achieved
  
💸 Expense KPI
- Expense Actual
- Expense Budget
- Expense Variance %
- Percentage of Expense Budget Utilized
  
📈 Income Trend Analysis
  Compares yearly Income Actual and Income Budget to identify financial performance trends over time.

📉 Expense Trend Analysis
  Displays yearly Expense Actual and Expense Budget to evaluate spending patterns and cost control.

📋 Income Performance Table
  Provides project-wise comparison of:
- Income Actual
- Income Budget
- Income Variance %
- Budget Achievement %
  Conditional formatting highlights projects performing above or below budget.

📋 Expense Performance Table
  Shows project-wise:
- Expense Actual
- Expense Budget
- Expense Variance %
- Budget Utilization %
  Color indicators help quickly identify projects requiring attention.

🎛️ Interactive Filters
   Users can filter the dashboard by:
- Year
- Month
- Project ID
  Allowing detailed analysis for individual projects or specific reporting periods.

4️⃣ Business Insights
